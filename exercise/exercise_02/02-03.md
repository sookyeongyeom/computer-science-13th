# 2.3 연습문제

## 1번
1회  
인출 : 명령레지스터1402 프로그램카운터02  
해석 : 02번 셀의 내용을 4번 레지스터에 저장  
실행 : 4번레지스터34

2회   
인출 : 명령레지스터3417 프로그램카운터04   
해석 : 4번 레지스터의 내용을 17번 셀에 저장   
실행 : 17번셀34

3회   
인출 : 명령레지스터C000 프로그램카운터06   
해석 : 종료   
실행 : 종료

17번 셀 = 34 = 0011 0100

답은 0011 0100이다.
